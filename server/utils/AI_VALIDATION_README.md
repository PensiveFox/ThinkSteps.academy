# AI-валидация решений студентов

Система использует локальную AI модель (Ollama) для умной проверки решений студентов.

## ✅ Что установлено

- **Ollama**: Локальный AI сервер
- **Модель**: llama3.2:3b (быстрая, 2GB)
- **Статус**: Работает на http://localhost:11434

## 🚀 Как это работает

1. **Студент отправляет решение**
2. **AI анализирует**:
   - Сравнивает с правильным решением
   - Находит ошибки на конкретных шагах
   - Генерирует понятный feedback
   - Дает подсказки без прямых ответов
3. **Студент получает детальный разбор**

## 📝 Пример feedback от AI

**Вместо:**
```
❌ Неверно. Ожидается: (4 - b) × 16, Получено: (4 - b) × 8
```

**AI даст:**
```
✓ Вы правильно начали: 4 - b
✓ Первое умножение верное: (4 - b) × 4

❌ Ошибка на шаге 3:
Вы написали: (4 - b) × 8
Но команда 1 означает "умножить на 4", а не на 2.

💡 Подсказка: Проверьте, сколько раз вы применили команду 1.
Результат предыдущего шага нужно умножить на 4.
```

## ⚙️ Настройка (опционально)

### Переменные окружения (.env)

```bash
# Ollama (по умолчанию включено)
USE_OLLAMA=true
OLLAMA_URL=http://localhost:11434

# Groq API (fallback, если Ollama недоступен)
GROQ_API_KEY=your_key_here
```

### Альтернативные модели

Если хотите другую модель:

```bash
# Более умная, но медленнее (7GB)
ollama pull llama3.1:8b

# Русскоязычная модель
ollama pull saiga

# Очень быстрая (1.5GB)
ollama pull phi3:mini
```

Затем в `aiValidator.ts` измените:
```typescript
model: 'llama3.1:8b' // или другая модель
```

## 🔧 Команды Ollama

```bash
# Проверить статус
ollama list

# Запустить модель вручную
ollama run llama3.2:3b

# Остановить сервис
sudo systemctl stop ollama

# Запустить сервис
sudo systemctl start ollama

# Удалить модель
ollama rm llama3.2:3b
```

## 📊 Производительность

- **Скорость**: ~50-100 токенов/сек (на GPU)
- **Память**: ~3GB RAM
- **Латентность**: 1-3 секунды на ответ
- **Стоимость**: 0₽ (полностью бесплатно)

## 🎯 Преимущества локальной AI

✅ Полностью бесплатно
✅ Работает без интернета
✅ Приватность данных студентов
✅ Неограниченное количество запросов
✅ Быстрая скорость ответа

## 🔄 Fallback система

Если Ollama недоступен, система автоматически переключится на:
1. **Groq API** (если есть ключ) - бесплатный облачный
2. **Базовую проверку** - строгое сравнение

## 📝 Интеграция в код

AI-валидация автоматически используется в:
- `validateExecutorFindB()` - проверка задач с исполнителями
- Можно добавить в другие типы заданий

Пример использования:
```typescript
import { validateWithAI } from '../utils/aiValidator'

const aiResult = await validateWithAI({
  userSolution: userCode,
  expectedSolution: correctSolution,
  problemDescription: taskDescription,
  userSteps: parsedUserSteps,
  expectedSteps: correctSteps,
})

// aiResult.feedback - детальный разбор от AI
```
