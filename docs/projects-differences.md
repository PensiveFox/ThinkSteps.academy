# Отличия между проектами до объединения

## ThinkSteps.academy (origin/main)

### 🎯 Назначение
- **Образовательная платформа** для подготовки к ОГЭ
- Материалы по математике и информатике
- Учебные планы и методические материалы

### 📁 Структура
```
wiki/docs/
├── матем/ya/           # Математика Я-класс
├── информатика/fipi/   # Задания ФИПИ
├── oge-*              # Интерактивные материалы ОГЭ
└── план/              # Учебные планы
```

### 🛠 Технологии
- Next.js + React
- Markdown для контента
- Prisma для данных
- Статичные HTML материалы

### 📊 Содержимое
- **240+ файлов** с учебными материалами
- **130k+ строк** контента
- PDF материалы, задания, планы
- Интерактивные таблицы ОГЭ

### 🚨 Проблемы
- ❌ 180 TypeScript ошибок
- ❌ Pre-push хуки failing
- ❌ Production build failing

## Почему объединение было необходимо

### 🎯 Цель объединения
1. **Сохранить образовательные материалы** из ThinkSteps.academy
2. **Добавить AI функционал** из reshai.space
3. **Создать единую платформу** для образования + автоматизации
4. **Исправить критические ошибки** в процессе

### 🔄 Результат объединения
```
Объединенный проект:
├── Образовательные материалы (из ThinkSteps.academy)
├── AI workflows (из reshai.space)
├── Чистая архитектура (из ThinkSteps.academy_new)
└── Стабилизация (новая ветка)
```

### 📈 Преимущества объединенного проекта
- **Полный функционал** - образование + AI
- **База знаний** - все материалы в одном месте
- **Автоматизация** - workflows для образовательных процессов
- **Масштабируемость** - готовая инфраструктура
