# Я-8 — типы заданий

| № задачи | Тип |
|---:|---|
| 1–10 | Вычисление выражений со степенями: подстановка значений и возведение в степень |
| 11–20 | Вычисление выражений со степенями (продолжение) — разные показатели и основания |
| 21–30 | Вычисление выражений со степенями (продолжение) — смешанные степени |
| 31 | Корни из произведения степеней: √(x^n · y^m) при заданных x, y |
| 32 | Корни из дробей со степенями: √(x^n / y^m) или √(a·x^n / y^m) |
| 33 | Формулы сокращённого умножения под корнем: √((a±b)²) = |a±b|, дробные значения |
| 34 | Формулы сокращённого умножения под корнем: √((a±b)²) с целыми и отрицательными значениями |
