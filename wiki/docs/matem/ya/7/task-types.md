# Я-7 — типы заданий

| № задачи | Тип |
|---:|---|
| 1 | Знаки выражений на координатной прямой: по положению x, y определить неверное утверждение (x+y, x−y, xy) |
| 2 | Дроби на координатной прямой: определить, какая дробь a/b отмечена точкой (сравнение дробей) |
| 3 | Оценка дроби: между какими целыми числами лежит число вида a/b |
| 4 | Принадлежность промежутку: выбрать промежуток, которому принадлежит число (интервалы/неравенства) |
| 5 | Сопоставление точек и десятичных чисел: точки A,B,C,D ↔ числа (порядок на прямой) |
| 6 | Число между двумя значениями: выбрать число, заключённое между заданными числами |
| 7 | Оценка квадратного корня: между какими числами заключено число √n |
| 8 | Дроби на координатной прямой (другой набор): определить дробь a/b по точке |
| 9 | Принадлежность отрезку [a; b]: выбрать число, принадлежащее отрезку |
| 10 | Число на координатной прямой: выбор по неравенствам/границам |
| 11 | Принадлежность отрезку [a; b] (ещё подборки): выбор числа из отрезка |
| 12 | Сопоставление точек и десятичных чисел (ещё набор): точка ↔ число |
| 13 | Корни √n на координатной прямой: выбор какого из √n отмечен точкой (сравнение подкоренных) |
| 14 | Рациональные числа (дроби) на прямой: оценка дроби, сравнение с целыми, привязка к точке |
