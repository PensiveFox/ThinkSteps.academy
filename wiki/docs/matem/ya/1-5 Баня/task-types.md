# Я-1-5 Баня — типы заданий

| № задания | Тип |
|---:|---|
| 1 | Соответствие между характеристиками печей (масса, стоимость, объем) |
| 2 | Вычисление геометрических параметров бани (объем, площадь стен, потолка, пола) |
| 3 | Сравнение стоимости покупки печей (с установкой, доставкой) |
| 4 | Расчеты со скидками и условиями покупки |
| 5 | Нахождение радиуса арки по чертежу |
