# Я-9 — типы заданий

| № задачи | Тип |
|---:|---|
| 1 | Линейные уравнения: перенос слагаемых (ax + b = cx) |
| 2 | Линейные уравнения: перенос с обеих сторон (ax + b = cx + d) |
| 3 | Линейные уравнения со скобками: a(x ± b) = c |
| 4 | Линейные уравнения с дробями: x ± x/a = b |
| 5 | Рациональные уравнения: дробь с переменной в знаменателе k/(x±a) = b |
| 6 | Неполные квадратные уравнения: x² = ax |
| 7 | Неполные квадратные уравнения: ax² ± bx = 0 (вынесение x за скобки) |
| 8 | Квадратные уравнения разность квадратов: x² − a = 0 |
| 9 | Квадратные уравнения с квадратами: (x+a)² = (x+b)² |
| 10 | Квадратные уравнения произведение: (ax+b)(cx+d) = 0 |
| 11 | Квадратные уравнения по Виету: x² + bx + c = 0 |
| 12 | Квадратные уравнения (приведение): x² ± bx = c |
| 13 | Квадратные уравнения общий вид: ax² + bx + c = 0 |
