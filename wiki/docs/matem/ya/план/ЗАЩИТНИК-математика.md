# ЗАЩИТНИК — Математическая композиция

## 📐 Общее описание
Слово "ЗАЩИТНИК" построено из графиков математических функций в стиле Desmos/GeoGebra. Чёрные тонкие линии на светлом фоне с координатной сеткой.

---

## 🔤 Буква З

```
y = -0.5x + 3,      -2 ≤ x ≤ 0
y = 0.5x + 1,       0 ≤ x ≤ 2  
y = -0.5x + 1,      0 ≤ x ≤ 2
```

*Описание: Верхняя и нижняя полуокружности соединены диагональными линиями*

---

## 🔤 Буква А

```
y = -2x + 6,        -1 ≤ x ≤ 1
y = 2x + 2,         -1 ≤ x ≤ 1
y = 4,              -0.5 ≤ x ≤ 0.5
```

*Описание: Две наклонные линии образуют треугольник с горизонтальной перекладиной*

---

## 🔤 Буква Щ

```
y = 6,              0 ≤ x ≤ 1
y = 0,              0 ≤ x ≤ 1
y = 6,              2 ≤ x ≤ 3
y = 0,              2 ≤ x ≤ 3
y = 6,              4 ≤ x ≤ 5
y = 0,              4 ≤ x ≤ 5
y = 3x,             5 ≤ x ≤ 6
y = -3x + 18,       5 ≤ x ≤ 6
```

*Описание: Три вертикальные линии с диагональными "хвостами" справа*

---

## 🔤 Буква И

```
y = 6,              0 ≤ x ≤ 0.5
y = 0,              0 ≤ x ≤ 0.5
y = -2x + 7,        0.5 ≤ x ≤ 2
y = 2x - 1,         0.5 ≤ x ≤ 2
y = 6,              2 ≤ x ≤ 2.5
y = 0,              2 ≤ x ≤ 2.5
```

*Описание: Две вертикальные линии соединены диагональным крестом*

---

## 🔤 Буква Т

```
y = 6,              -1 ≤ x ≤ 3
y = -x + 1,         1 ≤ x ≤ 3
```

*Описание: Горизонтальная линия с наклонной опорой справа*

---

## 🔤 Буква Н

```
y = 6,              0 ≤ x ≤ 0.5
y = 0,              0 ≤ x ≤ 0.5
y = 6,              2.5 ≤ x ≤ 3
y = 0,              2.5 ≤ x ≤ 3
y = 2.4x,           0.5 ≤ x ≤ 2.5
```

*Описание: Две вертикальные линии соединены диагональю*

---

## 🔤 Буква И (вторая)

```
y = 6,              0 ≤ x ≤ 0.5

y = 2x - 1,         0.5 ≤ x ≤ 2
```

## 🔤 Буква К

```

y = 1.5x,           0.5 ≤ x ≤ 2
```

*Описание: Вертикальная линия с двумя диагоналями, образующими угол*

## 📊 Параметры отображения

### Стиль:
- **Цвет линий:** чёрный (#000000)
- **Толщина линий:** 1-2px
- **Фон:** белый (#FFFFFF)
- **Сетка:** светло-серая, шаг 1 единица

### Масштаб:
- **X:** от -2 до 25
- **Y:** от -1 до 8

### Расположение букв (по центру каждой буквы):
- **З:** x = 1
- **А:** x = 4  
- **Щ:** x = 8
- **И:** x = 13
- **Т:** x = 17
- **Н:** x = 20
- **И:** x = 23
- **К:** x = 26

---

## 🎨 Эстетические особенности

1. **Минимализм:** Только необходимые линии без украшений
2. **Математическая точность:** Все элементы построены функциями
3. **Читаемость:** Буквы легко узнаваемы
4. **Единый стиль:** Одинаковая толщина и цвет линий
5. **Интеллектуальная эстетика:** Строгий геометрический подход

---

## 🔧 Техническая реализация

Для построения в Desmos или GeoGebra:
1. Ввести каждую функцию с ограничениями области определения
2. Установить чёрный цвет линий
3. Настроить белый фон и сетку
4. Масштабировать для оптимального отображения

*Композиция демонстрирует красоту математики через создание осмысленного текста из чистых функций.*
