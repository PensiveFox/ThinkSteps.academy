# Стратегии развития и учебные треки

## Концепция (вдохновлено freecode.academy)

На freecode.academy есть **стратегии развития** (Growth Strategies) - это готовые треки обучения с набором технологий и заданий.

**Пример трека:**
```
Front-end Web Developer Junior
├── HTML
├── CSS
├── TypeScript
├── git
├── Styled-Components
├── Javascript
├── moment-js
├── npm
└── yarn
```

## Адаптация для ThinkSteps.academy

### Стратегия развития = Трек подготовки к ОГЭ

Вместо технологий - **типы заданий ОГЭ** с пошаговой проверкой.

## Структура каждого задания в треке

### Единая структура (как в вашем файле 02-архитектура-и-проверка.md)

```
📝 Задание: Исполнитель Вычислитель

1. 🎥 Короткое видео (3-5 минут)
   └── Объяснение метода решения

2. 📋 Алгоритм (чеклист)
   ├── Шаг 1: Записать команды как функции
   ├── Шаг 2: Пройти программу без вычислений
   ├── Шаг 3: Составить уравнение
   └── Шаг 4: Решить уравнение

3. ✍️ Выполнение
   └── Интерактивный конструктор выражений

4. 🤖 Пошаговая проверка
   ├── ✅ Шаг 1: Команды записаны правильно
   ├── ✅ Шаг 2: Выражение построено верно
   ├── ❌ Шаг 3: Уравнение составлено неверно
   └── 💡 Подсказка: "Приравняй итоговое выражение к 28"

5. 🔁 Исправление
   └── Возврат к проблемному шагу
```

## Система разблокировки

### Последовательное открытие

**Правило:**
- Следующее задание открывается только после успешного выполнения предыдущего
- Исключение: можно пропустить 1 задание в блоке (если совсем не получается)

**Пример:**
```
✅ Задание 1 → Задание 2 открыто
⏳ Задание 2 → Задание 3 заблокировано
🔓 Кнопка "Пропустить" (доступна после 3 попыток)
```

### Гибкая разблокировка (для преподавателя)

Преподаватель может:
- Открыть любое задание конкретному ученику
- Создать индивидуальный трек
- Изменить порядок заданий

## Преимущества системы треков

### Для ученика
- Понятный путь обучения
- Видит прогресс
- Мотивация через разблокировку
- Не теряется в заданиях

### Для преподавателя
- Структурированная программа
- Видит, где ученик застрял
- Может корректировать трек
- Готовые наборы заданий

### Для платформы
- Легко добавлять новые треки
- Можно продавать премиум-треки
- Аналитика по эффективности треков
- Геймификация через прогресс

### Фаза 2: Множественные треки
- 3-4 трека разной сложности
- Диагностика для подбора трека

### Фаза 3: Адаптивные треки
- Персонализация под ученика
- Автоматическая корректировка
