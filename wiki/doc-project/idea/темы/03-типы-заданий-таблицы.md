# Таблицы (Excel / Google Sheets / web-таблица)

## Как автоматизировать

НЕ даёшь настоящий Excel, а:
- либо web-таблицу (React + grid)
- либо импорт .xlsx / .ods

## Проверки

### 1. Проверка структуры
- Количество строк / столбцов
- Названия колонок
- Порядок

### 2. Проверка ячеек
- Точное значение
- **Формула, а не число** ❗
- Относительные / абсолютные ссылки

Примеры:
```javascript
assert(cell("B2").formula.includes("SUM"))
assert(cell("C4").value === 125)
```

### 3. Проверка логики

Например:
- Значение в C5 должно быть результатом вычисления, а не вписано вручную
- ➡️ проверяется через AST формулы

## Работа на мобилке

### Табличный конструктор (не Excel)

На мобилке:
- Тап по ячейке → появляется:
  - число
  - формула (упрощённая)
  - выбрать из списка функций

- Диапазон выделяется:
  - долгим тапом
  - или "выбери от A2 до B6"

❗ Никакого drag'а мышью.

### Проверка
- Такая же, как на ПК
- Просто данные уже у тебя в JSON

## Примеры объяснения ошибок

### 🧮 Формула

**Плохо:**
> Неверное значение

**Хорошо:**
> В ячейке C4 записано число, но по условию там должна быть формула
> 
> 💡 Подсказка: Используй формулу сложения значений из столбца B

## ODS - технические детали

ODS — это ZIP + XML.

Внутри есть:
- таблицы
- формулы
- диаграммы (как объекты)

Можно достать всё, что проверяет эксперт ОГЭ.
Да, грязно. Да, XML-ад.
Но зато идеальный контроль и объективность.
