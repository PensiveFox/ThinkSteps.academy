# Python — программирование

## Общий подход

Делаешь exactly как freeCodeCamp, но:
- с русскими задачами
- под ОГЭ
- с видео перед каждым этапом

## Проверка

### 1. Синтаксис

### 2. Использование нужных конструкций
- for, if, while

### 3. Запрет "читов"
- нельзя использовать sum(), если по заданию цикл

### 4. Тесты
```python
assert solve(5) == 15
```

### 5. Проверка алгоритма (AST-анализ)
```python
assert "for" in ast
```

## Пошаговая проверка

Добавь пошаговую проверку, а не "прошло / не прошло".

Типа:
- ✔ Таблица создана
- ✔ Формулы есть
- ❌ Диаграмма использует неверный диапазон
- 💡 Подсказка: "диаграмма должна строиться по столбцам A и B"
