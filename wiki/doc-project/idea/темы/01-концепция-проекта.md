# Концепция проекта ThinkSteps.academy

## Основная идея

Веб-интерфейс для подготовки учеников 9 классов к ОГЭ по информатике (позже математика).

### Ключевые принципы

**Проверять не результат, а выполнение правил (алгоритма)**
- Как линтер + автотесты + инспектор структуры
- Система не спрашивает: ❌ «Какой ответ?»
- А спрашивает: ✅ «Ты сделал это так, как требует задание ОГЭ?»

### Название проекта

**ThinkSteps.academy** - выбрано как финальное название

Альтернативы рассматривались:
- FreeThink.academy
- MindSteps.academy
- AlgoSteps
- StepAcademy
- LogicSteps

**Слоган:**
- "Учим думать, а не угадывать"
- "Алгоритмы шаг за шагом"
- "Подготовка к ОГЭ через мышление"
- "Не ответы, а решения"

### Целевая аудитория

- Ученики 9 классов
- Подготовка к ОГЭ по информатике
- Работа с мобильных устройств (mobile-first)

### Бизнес-модель

Проект является:
- Витриной экспертизы преподавателя
- Каналом продаж услуг (ОГЭ, группы, индивидуально)
- ЛК для взаимодействия с учениками
- Контент + автопроверка
- Комьюнити вокруг преподавателя

**Стратегия запуска:**
- Закрытая учебная платформа для своих учеников
- Доступ только по ссылке
- Использование iframe для встраивания видео других учителей
- Постепенная замена на собственный контент

### Ценностное предложение

ThinkSteps.academy — это закрытая учебная платформа для учеников, где мы учимся решать задачи ОГЭ шаг за шагом, а не просто подбирать ответы.

**Ценность не в видео, а в:**
- Алгоритмах
- Интерактиве
- Проверке
- Траектории обучения

Видео = ввод
ThinkSteps = работа мозгом
