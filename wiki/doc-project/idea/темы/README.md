## Ключевые концепции проекта

### Название
**ThinkSteps.academy** - "Учим думать, а не угадывать"

### Основная идея
Проверять не результат, а выполнение правил (алгоритма). Пошаговая проверка решений с обратной связью на каждом этапе.

### Целевая аудитория
- Ученики 9 классов
- Подготовка к ОГЭ по информатике
- Mobile-first подход

### Бизнес-модель
- Закрытая платформа для своих учеников
- Канал продаж услуг преподавателя
- ЛК для взаимодействия
- Использование iframe для встраивания обучающих видео

### Технический стек
- Frontend: Next.js 16 + React 18
- Backend: Express 4 + Apollo Server 5 + GraphQL
- Database: PostgreSQL + Prisma ORM 6
- Auth: JWT
- n8n: AI agents
