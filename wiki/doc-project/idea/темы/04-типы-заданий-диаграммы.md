# Диаграммы

## Что реально проверять

Да, ВСЁ, что требует ОГЭ, можно автоматизировать:

- Тип диаграммы (круговая, столбчатая)
- Диапазон данных
- Начальная ячейка
- Подписи
- Легенда
- Проценты / значения

## Как технически

Диаграмма — это объект, а не картинка

Внутри:
```json
{
  "type": "pie",
  "dataRange": "A2:B6",
  "startCell": "D2",
  "labels": true
}
```

Если ученик:
- взял не тот диапазон → ❌
- построил столбчатую вместо круговой → ❌
- диаграмма "похожа", но логически не та → ❌

🔥 **Это то, где твой сервис будет сильнее школы.**

## Работа на мобилке

Парадоксально, но для мобилки это **идеально**.

### UI:

**Выбрал тип:**
- круговая
- столбчатая

**Выбрал данные (чекбоксы):**
- ☑ столбец A
- ☑ столбец B

**Нажал «Построить»**

### Проверка:
- тип диаграммы
- источник данных
- позиция (логическая, не пиксели)

**На мобилке даже проще, чем на ПК.**

## Примеры объяснения ошибок

### 📊 Диаграмма

> Диаграмма построена правильно по виду, но использует неверный источник данных
> 
> 💡 Подсказка: Диаграмма должна строиться по данным из диапазона A2:B6

### 📐 Расположение

> Диаграмма размещена не в той области листа
> 
> 💡 Подсказка: Перемести диаграмму так, чтобы её левый верхний угол начинался в ячейке D2
