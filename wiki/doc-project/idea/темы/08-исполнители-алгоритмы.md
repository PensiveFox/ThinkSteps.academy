# Исполнители и алгоритмы (1 часть ОГЭ)

## Ключевая проблема

На ОГЭ 1-я часть — бумага, голова и ручка.
Рисовать графы, таблицы истинности, схемы на телефоне — плохо.

## Решение

🎯 **Тренировать нужно не рисование, а принятие решений**

На экзамене ученик:
- рисует для себя
- а проверяют только итоговый выбор

Значит, в обучении:
- мы не повторяем форму экзамена
- мы тренируем мышление, которое к нему ведёт

## Общий принцип

Ученик выполняет МЫСЛЕННЫЕ шаги алгоритма,
а система проверяет, что шаги правильные.

Не «нарисуй», а:
- выбери
- отметь
- пройди
- зафиксируй

## Примеры заданий

### 🔹 Пример 1: Графы

**Как на экзамене:**
- рисует граф
- считает пути
- выбирает ответ

**Как в системе (с телефона):**

❌ «Нарисуй граф»
✅ Интерактивный граф-симулятор

**Вариант A (лучший):**
- вершины уже есть
- рёбра кликабельны (вкл / выкл)
- ученик:
  - включает нужные рёбра
  - отмечает путь
  - считает длину

**Система проверяет:**
- выбран ли корректный путь
- соответствует ли минимальности
- не нарушены ли условия

👉 рисовать не надо
👉 логика сохранена

**Вариант B (ещё проще):**
Вообще без графа

Ученик отвечает на шаги:
- Сколько вершин?
- Есть ли путь A → D без B?
- Минимальное число рёбер?

Это ровно те же мыслительные шаги, просто без карандаша.

### 🔹 Пример 2: Таблицы истинности / логика

**На бумаге:**
- рисуют таблицу
- вычисляют

**В системе:**
- таблица уже есть
- ученик:
  - выбирает значения
  - отмечает строки
  - кликает итог

**Проверка:**
- по шагам
- с подсветкой ошибки

👉 рисовать = не обязательно
👉 мыслить = обязательно

### 🔹 Пример 3: Кодирование, объекты, исполнители

Идеально ложится на мобилку:
- стрелочки
- шаги
- "выполни команду"
- "куда попадёт исполнитель"

**Тут телефон даже лучше бумаги.**

## Стратегия платформы

### До экзамена
- интерактив
- клики
- шаги
- алгоритмы

### Перед экзаменом
Отдельный режим: «как на экзамене»
- без подсказок
- без визуализации
- только вопрос → ответ

## Вывод

❌ Пытаться «перенести тетрадь в телефон» — ошибка
✅ Нужно перенести мышление в интерфейс

И тогда:
- мобилка ≠ проблема
- 1 часть ОГЭ автоматизируема
- подготовка становится сильнее, чем «просто писать»
