# Презентации

## Что можно проверять

Не идеально, но на уровне ОГЭ — более чем.

Проверяем:
- Количество слайдов
- Расположение элементов
- Примерный размер изображений
- Шрифт
- Размер текста
- Наличие заголовка

## Как технически

У презентации:
- DOM-структура
- bounding boxes элементов

```json
{
  "slide": 2,
  "textBlocks": [
    { "font": "Arial", "size": 18, "area": "center" }
  ],
  "images": [
    { "width": ">30%", "alignment": "right" }
  ]
}
```

⚠️ Не "пиксель в пиксель", а допуски, как у реального эксперта.

## Работа на мобилке

### Только шаблоны

❌ Свободный дизайн → ад
✅ Шаблон + правила

### Что даёшь ученику:

- 3–4 готовых макета слайда
- Фиксированные зоны:
  - заголовок
  - текст
  - изображение

### Ученик:
- вводит текст
- выбирает картинку
- меняет местами блоки

### Проверка:
- заполнены ли нужные зоны
- объём текста
- наличие изображения

**Для ОГЭ — более чем достаточно.**
