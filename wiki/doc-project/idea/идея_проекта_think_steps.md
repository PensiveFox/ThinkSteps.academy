# Идея проекта ThinkSteps.academy

## Общая концепция

ThinkSteps.academy — это **закрытая образовательная платформа** для учеников, занимающихся подготовкой к ОГЭ (в первую очередь по информатике), с упором на **алгоритмическое мышление и пошаговое решение задач**.

Проект изначально создаётся как **инструмент работы с собственными учениками**, доступ к которому предоставляется **по персональной ссылке**, а не как публичный массовый сервис.

---

## Ключевая идея

**Мы учим не ответам, а ходу решения.**

Ценность проекта не в видеоконтенте, а в:
- алгоритмах решения задач
- пошаговой практике
- автоматизированной проверке действий
- анализе ошибок учеников

Видео используются только как **вводный обучающий материал**.

---

## Использование видео

- Видео берутся из **открытых источников (YouTube)**
- Используется только формат **iframe (встраивание плеера)**
- Видео **не хранятся** на платформе
- Авторство сохраняется, видео остаётся на исходной платформе

Роль видео:
- объяснение темы
- демонстрация типового подхода

Основная работа ученика происходит **после просмотра видео**.

---

## Формат доступа

На старте платформа:
- **не публичная**
- без открытого каталога заданий
- используется только учениками преподавателя

Это снижает юридические риски и позволяет спокойно отладить методику.

---

## Формат заданий

- Все задания **авторские**
- Соответствуют **типам заданий ОГЭ (ФИПИ)**, но не копируют оригинальные формулировки
- Задания адаптированы под:
  - пошаговую проверку
  - работу с мобильных устройств

---

## Как выглядит учебный процесс для ученика

1. Просмотр короткого видео по теме (iframe)
2. Ознакомление с алгоритмом решения
3. Выполнение задания по шагам
4. Автоматическая проверка каждого шага
5. Подсказки и разбор ошибок
6. Фиксация прогресса в личном кабинете

---

## Роль платформы для преподавателя

ThinkSteps.academy позволяет преподавателю:
- видеть ход мыслей ученика, а не только ответ
- анализировать типовые ошибки
- экономить время на проверке
- использовать платформу как:
  - учебную среду
  - инструмент взаимодействия
  - канал записи на занятия

---

## Стратегия развития

1. Закрытое использование с собственными учениками
2. Отладка методики и интерфейса
3. Постепенное добавление собственного видеоконтента
4. Возможное расширение:
   - другие предметы
   - другие преподаватели
   - частично открытые курсы

---

## Краткое позиционирование

> ThinkSteps.academy — закрытая учебная платформа,
> где ученики учатся решать экзаменационные задачи
> **шаг за шагом**, а не угадывать ответы.
