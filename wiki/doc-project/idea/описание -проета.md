🎯 Цель: из 3 → 6
Работает так:

Применяем команду 2:

9² = 81

Теперь у нас: 81

Чтобы получить 8 из 81, достаточно взять команду 1 — зачеркнуть справа:
81 → 8

Далее: команда 2:
8² = 64

И снова команда 1:
64 → 6

Итого последовательность команд: 2 2 1 2 1

🔹 Чтобы получилось 6 после удаления, число должно быть 60–69.
🔹 Из диапазона 60–69 только 64 — это квадрат от числа 8.
🔹 Получить 8 можно из 81 после удаления правой цифры.
🔹 81 — это квадрат от 9, а 9 — квадрат от 3.

### Custom Nodes

bash
cd server/n8n/custom-nodes && npm run build


### Credentials

Before running, set up your API credentials in credentials/:

1. Create credentials/system/openrouter.json:
json
[
  {
    "id": "openrouter-cred",
    "name": "OpenRouter",
    "type": "openRouterApi",
    "data": { "apiKey": "sk-or-v1-xxx" }
  }
]


2. Create credentials/bootstrap.env:
N8N_BOOTSTRAP_OWNER_EMAIL=admin@example.com
N8N_BOOTSTRAP_OWNER_PASSWORD=YourPassword123!
N8N_BOOTSTRAP_OWNER_FIRSTNAME=Admin
N8N_BOOTSTRAP_OWNER_LASTNAME=User


See [credentials/README.md](credentials/README.md) for full documentation.
