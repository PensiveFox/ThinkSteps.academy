# ExecutorChallenge Component

Компонент для интерактивного решения задач типа "Исполнитель - найти b" с 3 блоками по аналогии с freecode.academy.

## Структура

Компонент состоит из 3 основных блоков:

### 1. ExecutorTheory (Теоретический блок)
- Описание исполнителя и его команд
- Условие задачи (программа, начальное и конечное значения)
- Инструкции по решению

### 2. ExecutorEditor (Редактор кода)
- Monaco Editor для ввода решения
- Поддержка Python (и других языков в будущем)
- Автоматическое извлечение значения `b` из кода

### 3. ExecutorOutput (Результат выполнения)
- Пошаговая трассировка выполнения программы
- Отображение правильности решения
- Детальный feedback

## Использование

```tsx
import { ExecutorChallenge } from 'src/components/ExecutorChallenge'

<ExecutorChallenge
  content={{
    executor: {
      name: "Вычислитель",
      commands: {
        "1": { op: "mul", k: 4 },
        "2": { op: "sub", var: "b" }
      }
    },
    program: "21122",
    start: 4,
    target: 28,
    unknowns: {
      b: { type: "natural" }
    }
  }}
  description="Описание задания"
  instructions="Инструкции по решению"
  onSubmit={async (answer) => {
    // Отправка на сервер
    return {
      isCorrect: true,
      feedback: "Верно!",
      trace: [...]
    }
  }}
/>
```

## Особенности

- **Универсальность**: Monaco Editor позволяет использовать любые языки программирования
- **Трассировка**: Пошаговое отображение выполнения программы исполнителя
- **Адаптивность**: Grid layout для корректного отображения на разных экранах
- **Интерактивность**: Кнопка "Проверить" с валидацией и feedback

## Файлы

- `index.tsx` - Основной компонент с layout
- `ExecutorTheory.tsx` - Блок теории и условия
- `ExecutorEditor.tsx` - Блок редактора кода
- `ExecutorOutput.tsx` - Блок результатов и трассировки

## Зависимости

- `@monaco-editor/react` - Monaco Editor для React 18
- `styled-components` - Стилизация компонентов
- `next/dynamic` - Динамическая загрузка Monaco Editor (SSR)
